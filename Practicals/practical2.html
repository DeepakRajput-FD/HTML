Doctype stands for Document Type.
It is a statement to declare the type of the document.
A doctype or document type declaration (DTD) is an instruction that tells the web browser about the markup language in which the current page is written.


It is an empty tag, for example, it just has an initial tag and no end tag.
They are always present inside the (Head) tag.
They are used by Web Browsers, Search Engines, and other Web Services to rank the web pages accordingly.
